Perguntas Frequentes

Respostas às dúvidas mais comuns

Utilizamos Supabase Auth para autenticação segura. Pode criar conta com email e password, ou usar login por código OTP enviado para o seu email. A verificação de email é obrigatória para maior segurança.

Suportamos PDF, DOCX, XLSX, PPTX, TXT, MD, HTML, CSV, JSON, e imagens (PNG, JPG). Para PDFs digitalizados, utilizamos modelos de visão para OCR automático.

Sim! O GodMode suporta Ollama para processamento 100% local. Os seus dados nunca saem do seu servidor. Recomendamos os modelos qwen3:14b para texto e snowflake-arctic-embed para embeddings.

Não. Os seus dados nunca são partilhados com terceiros. Se usar providers cloud (OpenAI, etc.), os dados são enviados apenas para processamento, conforme as políticas de cada provider. Com Ollama, tudo fica local.

Extraímos automaticamente: factos (categorizados), decisões (com responsável e data), riscos (com impacto e probabilidade), perguntas (com prioridade), ações (com responsável e prazo), pessoas e relações organizacionais.

Sim. Pode exportar em Markdown (SOURCE_OF_TRUTH.md), JSON para integração com outras ferramentas, ou PDF formatado. A API REST também permite acesso programático a todos os dados.
